# Raktárkezelő Rendszer (Backend Base)

Egy Laravel alapú raktárkezelő szoftver.

---

## Technológiai Stack

* **Keretrendszer:** Laravel 11.x 
* **PHP Verzió:** ^8.2 
* **Hitelesítés (Auth):** Laravel Fortify
* **Jogosultságkezelés:** Spatie Laravel-Permission
* **Adatbázis:** MySQL / MariaDB

---

## Főbb Funkciók (Készen állnak)

1.  **Biztonságos Autentikáció:** Bejelentkezés, kijelentkezés és session-kezelés a Laravel Fortify motorjával.
2.  **Törzsadatok kezelése:** Márka, kategória, áfakulcs, termékek létrehozása, módosítása.
3.  **Készletnyilvántartás:** Bevétel, kivétel, szállítólevél kezelés. Lezárás után sztornózás, termékeknként nyomonkövethetők a változások.
